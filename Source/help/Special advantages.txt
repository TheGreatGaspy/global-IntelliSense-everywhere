English: Special advantages (updated: 08.02.2018):

There are no conflicts with any hotkeys because the script 
does not use hotkeys;) Off course.

This means you can use multiple IntelliSense at the same time 
(those of your favorite editor and your own). Maybe you just want to add a few special entries to your favorite editor.

It works everywhere. You can easily reuse 
word lists by including them or adjusting your filter rules (for experts).

Creating wordlists is very easy. 
It is enough if you copy text (for example from Wikipedia). 
However, each entry should be in one line. Not multiple entries in one line.
You can make complicated entries (for experts) in which 
you can run quite normal Autohotkey source code there (for experts).


German: Besondere Vorteile(updated: 08.02.2018):

Es gibt keine Konfligte mit irgendwelchen Hotkeys, 
da das Script keine Hotkeys verwendet ;) Off course.

Das bedeutet du kannst gleichzeitig mehrere IntelliSense 
verwenden (die von deinem Lieblings-Editor und 
deine eigene zusätlich). 
Vielleicht möchtest du nur ein paar spezielle Einträge 
deinem Lieblingseditor hinzufügen.

Es funktioniert überall. Du kannst Wortlisten einfach wiederverwenden 
indem du diese 
includest (RegEx dort auch möglich) oder Deine Filter-Regeln anpasst (für Experten).

Das erstellen von Wordlisten ist sehr einfach. 
Es reicht wenn du Text kopierst (z.B. aus Wikipedia). 
Allerdings sollte jeder Eintrag in einer Zeile stehen. Nicht mehrere Einträge in einer Zeile.
Du kannst komplizierte Einträge machen (für Experten) in dem du 
ganz normalen Autohotkey Quelltext dort ausführen lässt (für Experten).

DE: Die Installation ist sehr einfach. Du musst nichts konfigurieren.

EN: The installation is very easy. You do not have to configure anything. 
Just start downoad and start.ahk. However, at the beginning only example word lists exist (thus almost empty). If you need help here to create a word list please read a bit in the documentation or ask.
